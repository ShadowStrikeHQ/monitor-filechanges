# monitor-filechanges
Monitors specified directories for file changes using the `watchdog` library.

## Install
`git clone https://github.com/ShadowStrikeHQ/monitor-filechanges`

## Usage
`./monitor-filechanges [params]`

## Parameters
- `-d`: Directory to monitor.
- `-l`: Logging level (e.g., DEBUG, INFO, WARNING).
